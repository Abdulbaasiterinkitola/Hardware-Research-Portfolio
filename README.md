# Hardware Engineering and Research Portfolio

This portfolio contains documentation and design files for my personal research and group hardware automation projects.

## Projects

A system designed to automate the cooling process for hogs, improving livestock welfare and farm efficiency.

**Contents:**
*   **Design Files:** Wokwi simulation files and Proteus schematics.
*   **Documentation:** Project design PDF (`design_project_326_2024_2025.pdf`).

### Automatic Solar Street Lighting System

An automated lighting solution leveraging solar energy for efficient street illumination.

**Contents:**
*   **Simulation:** LTSpice design files (`ltspice design.asc`).
*   **Documentation:** Comprehensive project report and presentation slides.

### ULES ARB Research Competition 2025

Research proposal addressing Nigeria's national grid stability issues using electrical engineering principles.

**Contents:**

*   **Presentation:** Finale slides.
