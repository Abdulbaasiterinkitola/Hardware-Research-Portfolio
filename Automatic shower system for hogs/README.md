# Automatic Shower System for Hogs

## Project Overview
This project involves the design and implementation of an automatic shower system for hogs. The system is designed to maintain optimal temperatures for livestock, promoting health and efficiency in farm management.

## Contents
- **Design Document**: `design_project_326_2024_2025.pdf` - Comprehensive report detailing the system's design, requirements, and implementation.
- **Schematics**: `Proteus schematics_.jpg` - Circuit diagram of the system.
- **Simulation**: `wokwi design/` and `wokwi design.zip` - Simulation files for the system logic (C++ based).

## Technologies
- **Simulation**: Proteus, Wokwi
- **Hardware**: Microcontroller-based automation
