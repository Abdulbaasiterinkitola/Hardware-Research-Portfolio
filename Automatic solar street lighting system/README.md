# Automatic Solar Street Lighting System

## Project Overview
This project details the design of an automated street lighting system powered by solar energy. The system aims to provide energy-efficient illumination that operates autonomously based on ambient light conditions.

## Contents
- **Project Report**: `solar lighting system report_.docx` - Full technical report on the system's design and performance.
- **Presentation**: `solar lighting system.pptx` - Slides summarizing the project for presentation.
- **Circuit Simulation**: `ltspice design.asc` - LTSpice simulation file for the power and control circuits.
- **Design Document**: `design_project_331_2024_2025_session.pdf`

## Technologies
- **Simulation**: LTSpice
- **Energy Source**: Solar Photovoltaics (PV)
- **Control**: Analog/Digital circuit logic for day/night detection
